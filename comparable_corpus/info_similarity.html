Une similarité est un évènement équivalent que l'onretrouve dans les différents textes comparés, équivalent d'un point de vue de partage de contenu narratif. Les similarités peuvent être consultées individuellement en cliquant sur une étiquette de similarité dans un texte. Chaque similarité est représentée avec un code couleur permettant de repérer chaque passage équivalent dans les différents textes.
